## API

### ProgressBar

| Name        | Description                  | Type     | Default   |
| ----------- | ---------------------------- | -------- | --------- |
| percent     | percent of the progress bar  | `number` | `0`       |
| strokeColor | color of the progress bar    | `string` | `#1677FF` |
| strokeWidth | line width, the unit is `px` | `number` | `3`       |
